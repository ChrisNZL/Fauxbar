Quick Delete
If Quick Delete is enabled and you press Delete:
 • any highlighted history results will be deleted from both Chrome and Fauxbar.
 • any highlighted search queries will be deleted from Fauxbar's query history.
Highlighted bookmarks remain safe and will not be deleted, though they may disappear from Fauxbar for a moment.

Keyboard Shortcut Options

You can customize various shortcuts to open Fauxbar with your keyboard.
Instructions
To view or modify Fauxbar's keyboard shortcuts:
  1. Open Chrome's Extensions page.
  2. Scroll to the bottom of the page.
  3. Click the "Keyboard shortcuts" link.
  4. Under Fauxbar, several shortcut options will be available.
  5. To modify a shortcut, click an input box, then press your desired keys.
Default Shortcuts
Activate the extension: (?)Not set
Open Fauxbar's Address Box in the current tab:Alt+D
Open Fauxbar's Address Box in a new tab:Ctrl+T
Open Fauxbar's Search Box in the current tab:Ctrl+K
Open Fauxbar's Search Box in a new tab:Ctrl+Shift+K

Memory
To minimize Fauxbar's memory usage, Fauxbar Memory Helper is a Chrome extension that quietly reloads Fauxbar when your computer has been idle for ~10 minutes, as long as none of Fauxbar's pages are open.
Errors

Hello!
Fauxbar has updated itself to version 1.5.0.

It's been almost 3 years since I last updated Fauxbar, so I thought I'd breathe some quality-of-life fixes and changes to get this extension working properly with the latest version of Chrome.

Changelog:
  • Added DuckDuckGo as a default search engine.
  • Updated Search Box favicons for Bing, Google, and Yahoo!.
  • Updated the default page icon in the Address Box to match Chrome's default Omnibox icon.
  • Increased padding on the Address Box and Search Box.
  • Increased padding on Address Box results.
  • Removed the option and functionality for "Show speech input options" because x-webkit-speech has been deprecated and no longer works.
  • Replaced Google Code links with GitHub links in the Fauxbar menubar menu.
  • Removed Facebook and Twitter links.
  • Removed the Donate button. (though if you'd like to throw a couple of dollars my way, please purchase a copy of my game and have some fun!)
  • Removed the Support section from the Options page. (the links to the GitHub Issues page from the Fauxbar menu is good enough)
  • Removed instructions to email error logs from the error log viewer. Please submit issues to the Issue Tracker if needed.
  • The Options page now refreshes itself if you change to a different tab and then switch back, in an effort to resolve an issue where images within the Options page become hidden for some reason I can't figure out.
  • Fixed an error occurring when attempting to create a notification when indexing was complete.
  • Fixed the Options page throwing an error.
  • Fixed an error occurring when clicking a number field on the Options page.
  • Fixed an issue where using a search engine in the Address Box would increase the Address Box's padding.
  • Fixed site tile titles that were too long sometimes not wrapping and increasing the tile's height.
  • Fixed Address Box and Search Box icons sometimes being hidden when returning back to an open Fauxbar tab.
